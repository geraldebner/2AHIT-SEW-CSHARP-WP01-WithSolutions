# CommandLine

## Aufgabenstellung

Name der Abgabedatei: *CommandLine.cs* 

Erstellen Sie folgende Funktionen *Calculator*, *InputStringPrint* und *InputDoublePrint*.  Die Funktionen diener zum Kennenlernen Konsoleneigabe und Konsolenausgabe von C#.



Die Funktion *Calculator* soll einen Taschenrechner mit Grundfunktionen implementieren. Dabei werden zwei Gleitkommezahlen und die Rechenoperation als Parameter übergeben. 
Das Ergebnis der Rechenoperation soll in der Konsole (Terminal) ausgegeben werden.

Implementieren sie die Rechnenoperationen: +, -, *, /

Die Ausgabe soll in folgendem Format erfolgen: *result: RECHENERGEBNIS* 

### Beispielausgabe 

a=5.0, b=2.0, operation='+'

```
result: 7.000
```

Die Funktion *InputStringPrint* soll eine Konsoleneingabe erfassen und wieder in der Konsole ausgeben.

Die Ausgabe soll in folgendem Format erfolgen: *input: EINGABE* 

### Beispielausgabe 


```
HALLO
input: HALLO
```

Die Funktion *InputDoublePrint* soll eine Konsoleneingabe einer Gleitkommazahl erfassen und wieder in der Konsole ausgeben.


### Beispielausgabe 


```
4.5
4.5
```

## Einschränkungen

Die Beispieldatei muss den Strukturangaben aus dem Unterricht folgen um Modultests zu ermöglichen.
